# Notificações

O Alpex Digital Hub possui um sistema de notificações em tempo real que mantém você informado sobre eventos importantes da plataforma.

## Acessando as Notificações

O ícone de **sino** na barra superior indica suas notificações. Quando há notificações não lidas, um indicador visual aparece sobre o ícone.

<Steps>
  <Step title="Clicar no ícone de sino">
    Na barra superior, clique no **ícone de sino** para abrir o painel de notificações.
  </Step>
  <Step title="Visualizar notificações">
    O painel exibe suas notificações mais recentes, ordenadas da mais nova para a mais antiga.
  </Step>
</Steps>

## Tipos de Notificações

O sistema envia notificações automáticas para diversos eventos, incluindo:

| Evento | Descrição |
|--------|-----------|
| **CTE inválido** | Quando um CTE importado apresenta inconsistência |
| **Pré-cotação criada** | Quando uma nova pré-cotação é registrada |
| **Cotação para aprovação** | Quando uma cotação está aguardando aprovação do gestor |
| **Coleta confirmada** | Quando uma coleta é confirmada |
| **Diferença no CTE** | Quando a diferença entre CTE e cotação aprovada excede R$ 0,50 |

## Gerenciar Notificações

### Marcar como Lida

Você pode marcar notificações como lidas de duas formas:

- **Individual** — Clique na notificação específica para marcá-la como lida
- **Todas de uma vez** — Clique no botão **Marcar todas como lidas** no topo do painel

### Tempo Relativo

As notificações exibem o tempo de forma relativa para facilitar a leitura:

- "há 2 minutos"
- "há 1 hora"
- "há 3 dias"

## Notificações em Tempo Real

O sistema utiliza tecnologia de push (Pusher) para entregar notificações em tempo real, sem necessidade de recarregar a página. Quando um evento relevante ocorre, a notificação aparece automaticamente no painel.

<Note>
  As notificações são específicas para cada usuário. Você só receberá notificações de eventos relacionados às suas permissões e responsabilidades.
</Note>
