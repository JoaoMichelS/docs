# Dashboard de Frete

O Dashboard de Frete oferece uma visão consolidada dos indicadores e métricas do módulo de gestão de frete, permitindo ao gestor acompanhar o desempenho logístico em tempo real.

## Acessar o Dashboard

No menu lateral, navegue até **100 - Gestão de Fretes > 112 - Dashboard**.

## KPIs e Métricas

O dashboard apresenta indicadores-chave de desempenho (KPIs) do módulo de frete:

| KPI | Descrição |
|-----|-----------|
| **Total de cotações** | Quantidade de cotações criadas no período |
| **Cotações aprovadas** | Quantidade de cotações aprovadas |
| **Cotações pendentes** | Quantidade aguardando aprovação |
| **Valor total de frete** | Soma dos valores de frete aprovados |
| **Diferença CTE** | Total de divergências entre CTEs e cotações |
| **Coletas realizadas** | Quantidade de coletas confirmadas |

## Gráficos e Relatórios

O dashboard pode incluir visualizações como:

- **Gráfico de cotações por período** — volume de cotações ao longo do tempo
- **Gráfico de frete por transportadora** — distribuição de valores por transportadora
- **Gráfico de frete por filial** — comparativo entre filiais
- **Gráfico de divergências** — CTEs com diferença de valor

## Filtros do Dashboard

| Filtro | Descrição |
|--------|-----------|
| **Período** | Selecionar intervalo de datas |
| **Filial** | Filtrar por filial específica |
| **Transportadora** | Filtrar por transportadora |

<Note>
  O dashboard é atualizado dinamicamente conforme os filtros selecionados. Utilize os filtros para analisar períodos específicos e comparar o desempenho entre filiais e transportadoras.
</Note>
