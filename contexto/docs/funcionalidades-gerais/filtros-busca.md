# Filtros e Busca

O Alpex Digital Hub oferece filtros avançados em todas as telas com tabelas de dados. Os filtros permitem localizar informações específicas rapidamente em grandes volumes de dados.

## Como Usar os Filtros

<Steps>
  <Step title="Abrir painel de filtros">
    Clique no botão **Filtros** (ícone de funil) localizado acima da tabela para expandir o painel de filtros.
  </Step>
  <Step title="Preencher os critérios">
    Preencha os campos de filtro desejados. Você pode combinar múltiplos filtros simultaneamente.
  </Step>
  <Step title="Aplicar">
    Clique em **Aplicar** ou **Buscar** para filtrar os dados. A tabela será atualizada automaticamente.
  </Step>
</Steps>

## Tipos de Filtros

| Tipo | Como Funciona | Exemplo |
|------|--------------|---------|
| **Texto** | Busca parcial (contém) | Digitar "SILVA" encontra "JOSE DA SILVA" |
| **Data (de/até)** | Intervalo de datas | De 01/01/2025 até 31/01/2025 |
| **Valor (de/até)** | Faixa de valores numéricos | De R$ 100,00 até R$ 500,00 |
| **Peso (de/até)** | Faixa de peso | De 100 Kg até 1.000 Kg |
| **Seleção** | Lista dropdown | Selecionar uma filial específica |
| **Multi-seleção** | Múltiplas opções | Selecionar vários estados |
| **Status** | Estado do registro | Pendente, Aprovado, Reprovado |

## Limpar Filtros

Para remover todos os filtros aplicados:

<Steps>
  <Step title="Clicar em Limpar">
    Clique no botão **Limpar Filtros** no painel de filtros para resetar todos os campos.
  </Step>
  <Step title="Verificar">
    A tabela voltará a exibir todos os registros sem restrição.
  </Step>
</Steps>

## Ordenação de Colunas

Além dos filtros, você pode ordenar os dados clicando no cabeçalho de qualquer coluna que tenha o ícone de ordenação:

- **Primeiro clique** — ordena em ordem crescente (A-Z, menor para maior)
- **Segundo clique** — ordena em ordem decrescente (Z-A, maior para menor)

<Note>
  Os filtros e a ordenação podem ser combinados livremente. Por exemplo, você pode filtrar por uma filial específica e depois ordenar por valor decrescente para ver os maiores fretes primeiro.
</Note>
