# Exportação de Dados

O Alpex Digital Hub permite exportar dados de diversas telas para formatos externos, facilitando análises, relatórios e compartilhamento de informações.

## Formatos Disponíveis

| Formato | Disponível em | Descrição |
|---------|--------------|-----------|
| **Excel (.xlsx)** | Todas as telas com tabela | Planilha com dados filtrados |
| **PDF** | Coletas | Documento formatado para impressão |

## Exportar para Excel

<Steps>
  <Step title="Aplicar filtros (opcional)">
    Antes de exportar, aplique os filtros desejados para incluir apenas os dados relevantes na exportação.
  </Step>
  <Step title="Clicar em Exportar">
    Clique no botão **Exportar Excel** (ícone de planilha) localizado acima da tabela.
  </Step>
  <Step title="Aguardar download">
    O sistema gerará a planilha e iniciará o download automaticamente.
  </Step>
</Steps>

## Telas com Exportação para Excel

| Tela | O que é exportado |
|------|-------------------|
| **Pré-Cotação** | Lista de pré-cotações com filtros aplicados |
| **Cotação Avulsa** | Pedidos e cotações avulsas |
| **Cotação por Carregamento** | Pedidos agrupados por carregamento |
| **Cotação Padrão** | Tabela de cotações padrão |
| **Aprovar Cotação** | Cotações pendentes/aprovadas/reprovadas |
| **Upload CTE** | CTEs importados |
| **Conferência CTE** | Relatório de conferência CTE vs cotação |
| **Lista de Coletas** | Coletas registradas |

## Exportar PDF de Coleta

<Steps>
  <Step title="Acessar lista de coletas">
    Navegue até **100 - Gestão de Fretes > Coletas > 111 - Lista de Coletas**.
  </Step>
  <Step title="Localizar a coleta">
    Encontre a coleta desejada na lista.
  </Step>
  <Step title="Clicar em PDF">
    Clique no botão **PDF** na coluna de ações para baixar o documento da coleta.
  </Step>
</Steps>

## Templates de Importação

Algumas telas oferecem templates (modelos) em Excel para importação de dados:

| Tela | Template |
|------|----------|
| **Pré-Cotação** | Template para importação de pré-cotações |
| **Cotação Padrão** | Template para importação de cotações padrão |

Para baixar um template, clique no botão **Download Template** na tela correspondente.

<Note>
  Os dados exportados respeitam os filtros aplicados na tela. Para exportar todos os dados, certifique-se de limpar os filtros antes de exportar.
</Note>
